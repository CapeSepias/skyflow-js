
__Prerequisites__

  

* Node > 12.x.x

* npm > 6.x.x

  

## Building the repo

  

```shell

npm run build 

```

  

## Building only types

  

```shell

npm run build:types

```

  

## Type-Checking the repo

  

```shell

npm run type-check

```

  
## Running the dev environment

  

```shell

npm run dev

```
